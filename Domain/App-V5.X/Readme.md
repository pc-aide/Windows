# APP-V5.X

## Application Virtualization
Available only: 
 * Windows 10
   * Enterprise 
   * Education
 * Windows 8
 * Windows 7

## Enalbe Feature
````PowerShell
#Disabled default
Enable-appv
````
[<img src="https://i.ibb.co/8b3mdQd/image.png">](https://i.ibb.co/8b3mdQd/image.png)

---

## Add *.appv
````powershell
Add-AppvClientPackage <*.appv> | Publish-AppvClientPackage -Global
````
